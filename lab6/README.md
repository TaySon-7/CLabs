# Лабораторная работа №6.
## Bash.
## Задача:
Написать скрипт для удаления всех синонимов указанного файла из указанного каталога и его поддиректорий, используя расстояние Левенштейна.
## Решение:
Создать функцию levenshtein(), выполнять подсчёт расстояния через алгоритм Вагнера-Фишера, использующий матрицу по стоимости выполнения операций: вставка, удаление и замена символа. Для удобства храним только предыдущую и текущую строки в двух массивах. Далее запускаем цикл обхода по всем файлам директории и её поддиректорий.
### Запуск программы:
Чтобы запустить скрипт нужно указать файл, по которому будет проверяться синонимичность, директории, где будет происходить удаление файлов, максимальное расстояние Левенштейна для определения синонима.
```terminaloutput
bash main.sh /tmp/lab6_test/original.txt /tmp/lab6_test 1
```
### Пример работы
```terminaloutput
bash main.sh /tmp/lab6_test/original.txt /tmp/lab6_test 1  
Удалён синоним: /tmp/lab6_test/sub/original2.txt, расстояние Левенштейна: 1
Удалён синоним: /tmp/lab6_test/origina1.txt, расстояние Левенштейна: 1
Удалён синоним: /tmp/lab6_test/original.tx, расстояние Левенштейна: 1
Всего удалено синонимов: 3
```
```terminaloutput
bash main.sh /tmp/lab6_test/original.txt /tmp/lab6_test 9
Удалён синоним: /tmp/lab6_test/sub/different.txt, расстояние Левенштейна: 9
Всего удалено синонимов: 1
```
