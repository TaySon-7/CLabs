# Лабораторная работа №2.
## Сортировка и поиск.
## Задача:
Реализовать программу для ввода таблицы вида ключ-значение и осуществлять следующие процедуры:
1. Сортировка вставкой таблицы относительно ключей.
2. Бинарный поиск в таблице по ключу
## Решение:
создано три файла со строками из стихотворения А. С. Пушкина "К Чаадаеву" в отсортированном, обратном и случайном порядке(реализован генератор случайного порядка). Ключи и значения хранятся раздельно, сделан CLI для бинарного поиска по ключу.
### Запуск программы:
Чтобы запустить встроенный CLI, используйте данную команду в папке lab2. При запуске CLI автоматически производится тестирование.
```terminaloutput
make main
```
Отдельный запуск тестов
```terminaloutput
make test
```
