# Лабораторная работа №4.
## Сбалансированные деревья.
## Задача:
Реализовать AVL-дерево для хранения и поиска элементов с уникальными строковыми ключами и вещественными значениями(ключ не более 6 символов) со следующими операциями:
1. Добавление узла к дереву по ключу с балансировкой.
2. Удаление узла из дерева по ключу ключу с балансировкой.
3. Печать дерева.
4. Поиск значения элемента по заданному ключу.
## Решение:
Реализованы функции добавления узла с балансировкой, удаления узла с балансировкой, вывода дерева в консоль, очистки дерева, реализован CLI и поддержка ввода из файла, где вывод отправляется в другой файл.
### Запуск программы:
Чтобы запустить встроенный CLI, используйте данную команду в папке lab4. При запуске CLI автоматически производится тестирование.
```terminaloutput
make main
```
Отдельный запуск тестов.
```terminaloutput
make test
```
Если вы хотите запустить работы программы из файла input.txt и настроить вывод в output.txt, то используйте следующую команду:
```terminaloutput
make main files="input.txt output.txt"
```
