# Лабораторная работа №3.
## Арифметические выражения в деревьях.
## Задача:
Реализовать программу для ввода инфиксной записи математического выражения с следующими действиями:
1. Перевод в обратную польскую нотацию.
2. Составление дерева арифметического выражения из обратной польской нотации.
3. Замена всех бинарных минусов на сложение с унарным минусом.
## Решение:
реализованы функции ввода строки,парсинга в ОПН, постиоения дерева, изменения дерева и перевода из дерево в инфиксную запись.
### Запуск программы:
Чтобы запустить встроенный CLI, используйте данную команду в папке lab3. При запуске CLI автоматически производится тестирование.
```terminaloutput
make main
```
Отдельный запуск тестов (при каждом запуске тестов генерируется новый файл для случайного порядка).
```terminaloutput
make test
```
